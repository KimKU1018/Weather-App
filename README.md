# Weather-App
날씨 앱입니다.

# 기능 상세
- 도시 이름을 입력하면 현재 날씨 정보를 가져와 표시
- 도시 이름을 잘못 입력하면 서버로부터 응답받은 에러 메세지가 알럿으로 표시

# 활용 기술
- Current Weather API
- URLSession
